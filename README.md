# FS_Intern_-_Uniplaces-backend

Property Management API

A Node.js and Express API for managing properties. 
Tested with Postman.

Setup

    Install dependencies: npm install
    Start the server: node server.js

Server runs on http://localhost:3000.
Endpoints

    Add Property:
    POST /properties

    Delete Property:
    DELETE /properties

    Get Properties:
    GET /properties
